# n8n-nodes-companyenrich

This is an n8n community node. It lets you use the [CompanyEnrich API](https://companyenrich.com) in your n8n workflows.

[n8n](https://n8n.io/) is a fair-code licensed workflow automation platform.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n documentation.

## Credentials

You need a CompanyEnrich API Key to use this node.
1. Sign up at [CompanyEnrich](https://companyenrich.com).
2. Get your API Key from the dashboard.
3. In n8n, create a new credential for **Company Enrich API** and paste your key.

## Operations

### Company
*   **Enrich**: Enrich company data using a domain name.
*   **Find Similar**: Find similar companies (lookalikes) based on a seed domain and various filters.
*   **Search**: Search for companies using advanced filters (Location, Industry, Revenue, etc.).
*   **Count**: Count the number of companies matching specific criteria.
*   **Get Usage**: Retrieve your current API usage stats.
